# Advanced Examples

This directory contains more complex example programs demonstrating integrated usage.

| Example | Description |
|---------|-------------|
| `attention_demo.c` | Multi-head attention forward pass with RoPE and causal masking |
| `inference_demo.c` | Autoregressive text generation with KV-cache |

## Building

Examples are built as part of the main CMake build. Look in `build/examples/Release/` after building.
