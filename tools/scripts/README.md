# Scripts

This directory will hold utility scripts for development, testing, and deployment.

## Planned Scripts

| Script | Purpose | Version |
|--------|---------|---------|
| `build.sh` / `build.ps1` | Cross-platform build helper | v0.5 |
| `run_tests.sh` | Run all test suites | v0.5 |
| `format_code.sh` | Apply clang-format across the tree | v0.5 |
| `coverage.sh` | Generate coverage reports | v1.0 |
| `profile.sh` | Run benchmarks and profile | v1.0 |
| `docker_build.sh` | Build Docker images | v1.0 |
| `bench_compare.sh` | Compare benchmark results against baselines | v2.0 |
